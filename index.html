EHR

Electronic Health Record solution on-the-go

What is this?

This is the GitHub page for the EHR project. Basically what it does is that it allows you to setup a mobile electronic medical record server on a single-board-computer, such as Raspberry Pi, and allows medical experts to access the data through mobile apps.

Who build this?

We are a team of final year project students from HKUST. Go to our website and know more about us!

Who is this for?

This solution is design for One-2-One in Cambodia, but it is completely open source for anyone interested.

How to access it?

We are also building mobile apps for the solution on iOS (coming soon) and Android.
